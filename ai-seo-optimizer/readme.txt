A plugin to optimize SEO using AI, generate blog content, and provide an SEO chatbot assistant, with Google Gemini API integration.

== Description ==

This plugin leverages the power of Google Gemini API to provide advanced SEO analysis,
AI-powered blog content generation, and an intelligent SEO chatbot assistant. It supports
both English and Thai languages.

Features include:
* AI-driven SEO analysis and recommendations.
* Automated blog post generation.
* Integrated image generation suggestions.
* SEO chatbot for quick assistance.
* Google Analytics integration (planned).
* Support for English and Thai.

== Installation ==

1. Upload the `ai-seo-optimizer` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure the plugin settings, including your Google Gemini API Key.
